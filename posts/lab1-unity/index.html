Lab 1: Setup Unity on Linux

Unity is one of the leading game development and design platform nowadays. Unity is not only free but also allows you to create cross-platform games, meaning you can very well develop for mobile phones, computer or virtual head sets like the Oculus Quest. The first thing to do to install unity is :

  1. create an account and register to a licence.

    This is done directly on the unity website. When you click on get started it will directly go to the licence page with the big numbers. But don’t you worry, if you are using unity for your onwn use you don’t have to pay anything, you only have to clik on the individual tab to access the right licenses. I have personnaly chosen the Personal one, but you can choose the Student one which will probably ask you some kind of additional certificate.

    Once this is done you can either choose the guided path or the standard one. Now accept the terms and condition and unity hub is now downloaded onto your computer. But here you did not create an account which will come in handy once you open unity hub. Go back to unity website and click on the user icon and selec “Create a unity ID”. Now you just have to fill up the information to create your account.

  2. Launch UnityHub

    Now that you’ve downloaded unity Hub you have to launch it. On Linux, it saves a “AppImage”, you can easily launch it in command line:

    $ ./Downloads/UnityHub.AppImage

    When you open Unity Hub you reach the following view:

    You see that you need a licence however you can not activate a new one. This is because you need to log in first. Click on the user icon on the top right and enter you credentials previously created.

    Now that you’ve logged in the ‘activate new license’ button is now clickable ! Click on it and choose ‘Unity Personal’ and ‘I don’t use unity in a profesional capacity’ (if this is your case of course). There you have it your license is activated!

  3. Install unity versions

    Now if you get out of the settings, go to project and click on new, you see a message pop out at the bottom prevening you for creating anything. Indeed you need to download installs, which refers to particular unity versions.

    For this you need to go to the Install folder and click on Install (or locate if you already have one on your computer). Now you can choose the version you need. Be careful here: depending on what you will be doing on unity you will need certain modules, you can choose the one you desire when you click on next.

  4. Create your first project and start developing !

    Once the install is installed (this can take some time). You are now free to create your first project on the project folder simply by cliking “New”.
